data class ViewToImageConfig(view: View, backgroundColor: Int, padding: Padding)
Content copied to clipboard

Configuración para convertir una vista en una imagen.

Parameters

view

Vista que se va a convertir a imagen.

backgroundColor

Color de fondo a aplicar a la imagen, el predeterminado es transparente.

padding

Padding a aplicar entre la vista y los bordes de la imagen. Los valores se interpretan como dp, por ejemplo si el padding top = 10, al generar la imagen el padding top será de 10dp.

textWatermarkList

Lista de marcas de agua de texto, dejar la lista vacía para no aplicar ninguna.

drawableWatermarkList

Lista de marcas de agua de drawable, dejar la lista vacía para no aplicar ninguna.

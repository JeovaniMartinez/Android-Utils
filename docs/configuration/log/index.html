Log

The debug log can be globally enabled or disabled, and affects all utilities in the library. It is recommended to use the following configuration to enable it in development and disable it in production, it is only necessary to adjust it once within the app, either In the onCreate() of the singleton or main activity.

Base.logEnable = BuildConfig.DEBUG
note
Base is the superclass of most utilities.
